# Incrometales Web

Sitio web estático desarrollado para **Incrometales**, enfocado en mostrar productos, servicios y contenido informativo de la empresa.

---

## 🌐 Demo

Puedes ver el sitio en línea aquí:

👉 https://bycamiloam.github.io/Incrometales/

---

## 📁 Estructura del proyecto

El proyecto está construido con tecnologías web básicas:

- **HTML** → estructura de las páginas  
- **CSS** → estilos y diseño visual  
- **JavaScript** → interactividad  

Carpetas principales:

- `css/` → estilos  
- `js/` → scripts  
- `public/` → imágenes y recursos  

---
